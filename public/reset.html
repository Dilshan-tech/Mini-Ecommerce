Reset password

This page lets you change the saved demo password in this browser. It does not send any email or talk to a real database.

Back to login

How this works

The email and password are saved only in localStorage on this computer, not in a backend database. This keeps the project inside your CRUD syllabus.

  • Checks that the email matches the saved demo user
  • Updates the password stored in localStorage
  • Lets you log in again with the new password
