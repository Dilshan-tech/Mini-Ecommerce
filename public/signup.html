About this signup page

New users can create an account by providing their name, email and a strong password. The account is stored in this browser so that the login and reset password pages can recognise the same user.

  • Validates that all fields are filled
  • Prevents duplicate signup with the same email
  • Enforces a strong password with symbols, numbers and letters
