About this register page

This page lets a new shopper register with their basic details before using the store. It captures their name, email and a strong password so they can log in again later in the same browser.

  • Collects full name and contact email
  • Requires a strong password with symbols and numbers
  • Stores the demo user data safely in localStorage only
